This distribution of the noise library is intended for integration with BambuStudio.

The original source code was obtained from:
https://sourceforge.net/projects/libnoise/ (version 1.0)

Modifications made in this distribution:
- Removed the 'doc' directory
- Added CMake build support in the root directory

No changes have been made to the core source code of the library.
